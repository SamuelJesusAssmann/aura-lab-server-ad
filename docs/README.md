# Documentation

This directory contains the technical documentation for the Aura Lab Server AD project.

Each document describes the implementation and configuration of services within the lab environment.

## Covered Topics

* Hyper-V
* Windows Server
* DNS
* Active Directory Domain Services (AD DS)
* Organizational Units (OUs)
* Security Groups
* Domain Join
* Group Policy Objects (GPOs)
* DHCP
* File Server
* Home Folders

## Purpose

This documentation is intended to:

* Record deployment procedures
* Document configuration steps
* Provide implementation references
* Track the evolution of the lab environment
