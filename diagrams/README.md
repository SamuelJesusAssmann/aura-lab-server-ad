# Diagrams

This directory contains infrastructure and architecture diagrams used throughout the Aura Lab Server AD project.

The diagrams provide a visual representation of the lab environment, network topology, Active Directory structure, and service relationships.

## Contents

Diagrams may include:

* Network topology
* Active Directory structure
* Organizational Unit (OU) hierarchy
* Security Group relationships
* Group Policy architecture
* File Server structure
* Home Folder architecture

## Purpose

The purpose of these diagrams is to:

* Visualize the lab environment
* Simplify infrastructure documentation
* Support deployment and troubleshooting activities
* Improve understanding of system relationships
