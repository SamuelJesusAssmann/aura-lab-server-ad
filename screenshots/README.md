# Screenshots

This directory contains screenshots captured throughout the Aura Lab project.

The images stored here are used to support documentation, troubleshooting guides, and project implementation records.

## Purpose

* Document deployment steps
* Record configuration settings
* Illustrate troubleshooting procedures
* Provide visual references for lab architecture and services

## Organization

Screenshots should be grouped by topic whenever possible.

### Suggested Structure

```text
images/
├── active-directory/
├── dns/
├── domain-join/
├── gpo/
├── hyper-v/
├── file-server/
└── troubleshooting/
```

## Notes

* Remove any sensitive information before publishing screenshots.
* Use descriptive file names.
* Prefer PNG format for documentation clarity.

### Example File Names

```text
gpo-rdp-users-policy.png
domain-join-success.png
active-directory-ou-structure.png
dns-forward-lookup-zone.png
```
